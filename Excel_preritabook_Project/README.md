# 📊 Retail Sales Performance Dashboard – Excel Data Analytics Project

An end-to-end Excel Data Analytics project focused on analyzing retail sales transactions, customer behavior, product performance, and regional revenue trends using Pivot Tables, Charts, and Interactive Dashboards.

# 📂 Dataset Columns

| Column Name | Description |
|-------------|-------------|
| TransactionID | Unique transaction identifier |
| CustomerName | Name of the customer |
| Gender | Customer gender |
| Age | Customer age |
| Product | Product purchased |
| Category | Product category |
| Quantity | Number of units sold |
| UnitPrice | Price per unit |
| TotalPrice | Total amount before discount |
| DiscountPercent | Discount percentage applied |
| FinalPrice | Final amount after discount |
| Date | Transaction date |
| Region | Sales region |
| PaymentMode | Customer payment method |

# 📊 Analysis Performed

### 1️⃣ Sales Analysis
- Total sales performance.
- Revenue trend analysis.
- Product category comparison.

### 2️⃣ Customer Analysis
- Customer demographics by age and gender.
- Customer purchasing behavior.
- Top customers by spending.

### 3️⃣ Product Analysis
- Best-selling products.
- Category-wise revenue contribution.
- Quantity sold by category.

### 4️⃣ Regional Analysis
- Revenue by region.
- Sales distribution across regions.
- Regional performance comparison.

### 5️⃣ Payment Analysis
- Most preferred payment methods.
- Payment mode contribution to revenue.
